# react-native-rn-popup-modal

Popup and modal for rn projects

## Installation

```sh
npm install react-native-rn-popup-modal
```

## Usage

```js
import RnPopupModal from "react-native-rn-popup-modal";

// ...

const result = await RnPopupModal.multiply(3, 7);
```
